Dire Straits Song Matcher

Match each clue to the correct song title. This game keeps its HTML, CSS, and JavaScript in one file and adds a more interactive requirement to the project while staying connected to the Dire Straits scratch-page theme.

  • Each round gives one clue and four possible songs.
  • Your score updates immediately after each guess.
  • Use the reset button to reshuffle and play again.
